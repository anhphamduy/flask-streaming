Installation

  • Activate the virtual environment and install all the necessary packages.
  • Run python app.py in the console

Running

  • When starting the app, it should display the public host address of your computer.
  • By using that public host address, you could access the streaming video in any devices.

How does it work?

Python comes with a feauture called generators. By using generators, we are able to get all requests all process it one by one at the same time (in this case 250ms/pic). This is extremely important for streaming as order of events matters.

For the application, on the front end, images are captured 5 times per second, then will be sent to the server. The server will process each frame one by one then send back to all other clients
